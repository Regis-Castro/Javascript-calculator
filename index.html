Calculadora Javascript orientada a objetos

com prática de grid CSS e gradiente
Este projeto foi criado devido à seção 6 do curso "Orientação a objetos com Javascript (projetos e typescript)", do professor Matheus Battisti, da Udemy. O mesmo teve seu HTML e CSS modificados por mim para que eu pudesse praticar Grid CSS e gradiente combinado a uma imagem. Também houve alterações no arquivo Javascript e a calculadora precisaria receber mais algumas melhorias para funcionar perfeitamente, no entanto o foco foi apenas praticar um pouco de orientação a objetos. Todo o design foi desenvolvido em conjunto com Suelem Lamim, aplicando pela primeira vez meus conhecimentos aprendidos no curso "Design gráfico - fundamentos", de Margareth Domingues.
